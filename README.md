wtanaka.apache-kafka
====================

Installs Apache kafka

Requires that you have java installed already, e.g. using
https://galaxy.ansible.com/wtanaka/oracle-java/

Example Playbook
----------------

    - hosts: servers
      roles:
         - wtanaka.apache-kafka
